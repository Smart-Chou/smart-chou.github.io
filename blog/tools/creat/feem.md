::: tip Feem
Feem是一款跨平台的局域网传输工具
:::

## 它的优点是

- 轻量级
- 支持平台丰富
- 传输文件像聊天
- 支持在线和离线两种方式

## Feem支持的平台

- Android
- iOS
- Windows
- Windows Phone
- macOS Linux

## 其他

Feem传输文件有点像聊天软件 就是加入群组随意发送 反正好用就对了
