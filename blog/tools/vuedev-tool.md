::: tip
开发vue的必备浏览器调试插件
:::

这个插件在chrome浏览器中用来调试vuejs程序的工具，可以说是必备的插件

这一节就是安装教程

下载插件地址[立即直达](https://pictureknow.com/extension?id=d50143a5f53d406dbe992277bfc90521l)

![#](https://cdn.jsdelivr.net/gh/Smart-Chou/webphoto/image/105753-654628.png)

下载完成后打开chrome浏览器的拓展中心

![#](https://cdn.jsdelivr.net/gh/Smart-Chou/webphoto/image/163432-658058.png)

然后开启开发者模式

![#](https://cdn.jsdelivr.net/gh/Smart-Chou/webphoto/image/105837-598467.png)

然后把刚刚下载的插件直接拖动到这个页面，点击添加，就完成了安装

![#](https://cdn.jsdelivr.net/gh/Smart-Chou/webphoto/image/163229-454061.png)

安装完成后右上角会有这个标志，这就安装完成

![#](https://cdn.jsdelivr.net/gh/Smart-Chou/webphoto/image/163652-691443.png)
