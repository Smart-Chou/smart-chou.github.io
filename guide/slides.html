# 渐变

Transition 可以通过配置中的 transition 选项全局设置，也可以通过在特定幻灯片添加 data-transition 属性局部设置.

可能的值:

  • none
  • fade
  • slide
  • convex
  • concave
  • zoom

--

# 渐变

# 过渡动画

你可以在相邻的幻灯片上添加 data-auto-animate 使相同的 HTML 元素产生过渡动画效果。

# 功能

# 缩放

按下 alt (Linux 上使用 ctrl) 的同时点击幻灯片的任何元素，即可以向此元素进行放大。

再次点击即可缩小。
